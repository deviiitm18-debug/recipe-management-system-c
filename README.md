# Recipe Management System (C)

A simple **Recipe Management System** built using the **C programming language**.  
This program allows users to add, search, view, and manage recipes using file handling.

---

## Features

- Add Recipe 1
- Add Recipe 2
- Add Recipe 3
- View All Recipes
- Search for a Recipe
- Delete a Recipe
- Clear All Recipes
- Exit the Program

---

## Technologies Used

- C Programming
- File Handling
- Structures

---

## How to Run the Program

1. Compile the program

```
gcc main.c -o recipe
```

2. Run the program

```
./recipe
```
